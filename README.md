# Taste&Test 

Taste&Test is a comprehensive Software Quality Assurance project designed to validate a corporate ecosystem. This repository contains all test artifacts, ranging from planning documents and manual test cases to full-scale API, database, and automated regression suites.

---

##  Tech Stack & Tools

* **Test Management & Documentation:** MS Word, MS Excel (Test Plans, Test Cases, Bug Reports)
* **API Testing:** Postman
* **Database Testing:** MySQL / SQL
* **Automation Testing Framework:** Java, Selenium WebDriver, TestNG, Maven

---


##  Testing Phases & Scope

### 1. Manual UI Testing

* Performed end-to-end functional verification based on core business requirements.
* Documented extensive test coverage and managed execution passes using structured test sheets.

### 2. API Testing

* Validated RESTful endpoints using Postman.
* Ensured proper status codes, payload structures, schema validations, and data persistence.

### 3. Database Testing

* Executed backend database testing to cross-verify:

    * Data integrity
    * Constraint behavior
    * Transaction logging
    * Data consistency

### 4. Test Automation 

* Building a scalable automated regression framework utilizing:

    * Java
    * Selenium WebDriver
    * TestNG
* Structured under a standard Maven architecture for optimized dependency management.
